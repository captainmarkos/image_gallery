* Helpful Hints *

To help you understand the photo gallery and to keep your pics organized, think of the gallery as a room filled with many collections where each collection has many albums. Then within each album there are many pictures. Have a look at the goldencoconut.us photo gallery to see how the gallery is organized.
  • Each user has a Photo Gallery in which:
    • Each gallery may have many collections.
    • Each collection may have many albums.
    • Each album contains many images.
    • An album can only belong to one collection in the gallery.
    • An image can only belong to one album in the collection.

  • First - create a collection in which you want to add photo albms.
  • Next - create an album. You upload you pics into the album for that collection to which it belongs.
  • The collection & album images are going to be resized to thumbnail size (100 x 75). So instead of uploading a huge image you will save yourself some time if you resize it before uploading.
  • Try to keep the number of images per album less than 50 (for performance and visitor experience reasons).
  • You can share the URL to your collection(s) and/or album(s) by copying the URL from your browsers address bar. Do this from the "Photo Gallery" itself and not from "Manage Photos". Same is true for images. You can also paste the URLs into Facebook.
  • Uploaded images will be resized to 1200 x 900 (only if they are larger).
  • When uploading multiple images, keep the image sizes in mind.
