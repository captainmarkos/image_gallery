* Helpful Hints *

Image Management - Images must belong to an album in which that album belongs to a collection. Therefore when you add an image you must select to which collection and album it will belong. When adding images, image details such as title and description are associated to the image.
  • To add an image:
    • First choose the collection.
    • Next choose the album to which the image(s) will belong.
    • Finally add image details such as title and description.
    • The image will be resized to 1200 x 900 (if needed).
  • When modifying image details you change which album it belongs to. However the collection may not be changed because to do so the album would need to be moved to that collection.
  • To change which collection an album belongs to, find the album details and click "Modify". From there you change the collection.
